# Sistema Control de Buses
Un sistema para la gestión para transporte público.

**Procesos abordados**
- Gestión flota de buses
- Gestión de personal
- Gestión de paradas
- Gestión de rutas
- Gestión de recorridos
- Gestión de incidentes en recorridos

**Prerequisitos**
- Java 17
- Spring Boot v4.0.7
- MySQL 8.0

**Variables de entorno**
- SERVER_PORT
- FRONTEND_PORT
- DATABASE_URL
- DATABASE_USER
- DATABASE_PASSWORD
- JWT_SECRET_KEY
- JWT_EXPIRATION_TIME
- RABBITMQ_HOST
- RABBITMQ_PORT
- RABBITMQ_USER
- RABBITMQ_PASSWORD

Este proyecto fue desarrollado por grupo de estudiantes en el año 2026.
